Que vous découvriez le service d'images ou que vous veniez d'autres outils, ce guide vous aidera à apprendre les outils de déploiement Windows et à configurer votre espace de travail de déploiement.

Comprendre les outils de déploiement et d'installation de Windows

Pour commencer, il faut remonter à l'époque de Windows Vista afin de comprendre son système d'installation et de déploiement. Ne vous inquiétez pas, ce n'est pas une longue section.

Avant ce système d'exploitation, le CD d'installation contenait tous les fichiers de l'installation de Windows de manière compressée, mais ne contenait pas une installation complète de Windows dans un seul fichier. Par exemple, chaque fichier de 3D Pinball était compressé dans ses propres fichiers. Cela signifiait que vous aviez besoin du CD pendant tout le processus d'installation, jusqu'à la fin. Si vous le retiriez pendant l'installation, le système vous demanderait de le réinsérer :

Les transferts constants depuis le CD vers le HDD de l'ordinateur faisaient également que l'installation prenait du temps.

Face à ces inconvénients, Microsoft a commencé à travailler sur un nouveau système d'installation pendant le développement de Windows Vista (nom de code Longhorn) qui permettrait d'installer Windows en moins de 15 minutes :

Myers dit que l'objectif est qu'une installation automatisée de Longhorn prenne moins de 15 minutes du début à la fin. Comparé au temps d'installation de 45 à 60 minutes pour Windows XP, c'est incroyable, bien que les pré-alphas de Longhorn soient encore assez lents.

— Paul Thurrott, "Longhorn Setup and Deployment Strategies"

Cela a donné naissance au format de fichier WIM, qui est un seul fichier contenant une ou plusieurs installations complètes de Windows. Cela signifie que l'ordinateur n'a besoin d'effectuer des transferts depuis le média d'installation vers le disque dur qu'une seule fois : lors de l'expansion de l'image.

Évidemment, comme vous travaillez avec de nouveaux formats et standards de déploiement, la technologie existante ne fonctionnera pas, donc Microsoft a inclus plusieurs outils de déploiement, de gestion et de maintenance dans le AIK de Windows Vista.

Pendant le développement de Windows 7, Microsoft a tenu compte des retours clients et a consolidé tous ces outils en une seule application intégrée au système d'exploitation. Appelé DISM, ce programme peut effectuer toutes les opérations trouvées dans les outils de Windows Vista, et bien plus, grâce aux améliorations apportées dans les versions ultérieures de Windows.

DISM est un utilitaire en ligne de commande dont la syntaxe peut vous sembler difficile. Cependant, il existe plusieurs interfaces utilisateur à votre disposition, et celle-ci en fait partie.

Ce que DISMTools vous apporte

En tant qu'interface pour DISM, DISMTools offre des fonctionnalités intéressantes :

  • C'est le premier UI basé sur des projets, permettant une organisation plus facile de votre travail
  • Il fournit des informations riches et rapides, et offre des outils qui vous aident dans vos tâches de gestion. Plus de détails sont disponibles à l'Étape 2 de cette visite
  • Il propose des outils intégrés pour le déploiement et les tests d'images. Plus d'informations sont disponibles à l'Étape 3 de cette visite

Dans cette étape, vous apprendrez les tâches de base.

Créer votre premier projet et monter votre première image Windows

Pour créer votre premier projet, cliquez simplement sur le lien "Nouveau projet..." sur l'écran d'accueil, ou allez dans Fichier > Nouveau projet...

La boîte de dialogue suivante s'affichera :

Saisissez le nom et l'emplacement de votre projet. Vous pouvez faire preuve de créativité pour le nom. Ensuite, cliquez sur OK.

Une fois le projet créé, vous verrez la vue du projet :

Avant de pouvoir faire quoi que ce soit avec votre projet fraîchement créé, vous devez monter une image Windows. Pour ce faire, cliquez sur le bouton Monter l'image.... Vous verrez la boîte de dialogue suivante :

Spécifiez une image Windows, l'emplacement où la monter, et l'index. Vous pouvez configurer des options supplémentaires, mais elles ne sont pas requises dans la plupart des cas.

Vous pouvez en savoir plus sur cette tâche dans la documentation d'aide

Comment obtenir une image Windows ?

Les images Windows se trouvent généralement dans le fichier install.wim, qui se situe dans le dossier sources de votre média d'installation Windows. Vous pouvez aussi les trouver dans le fichier install.esd, qui est une version compressée du fichier WIM. Ce fichier se trouve habituellement au même emplacement que le WIM.

Après l'avoir copié, indiquez-le dans la boîte de dialogue, comme ceci :
